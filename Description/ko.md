이 스크립트는 YouTube Music의 볼륨 조정을 위한 단축키를 추가합니다.

## 사용법

- `Alt + ↑`: 볼륨을 1% 증가
- `Alt + ↓`: 볼륨을 1% 감소
- `Ctrl + Alt + ↑`: 볼륨을 5% 증가
- `Ctrl + Alt + ↓`: 볼륨을 5% 감소
- `Ctrl + Shift + Alt + ↑`:
  - `0% ~ 49%`: 볼륨을 50%로 설정
  - `50%`: 볼륨을 100%로 설정
  - `51% ~ 100%`: 볼륨을 100%로 설정
- `Ctrl + Shift + Alt + ↓`:
  - `0% ~ 49%`: 볼륨을 0%로 설정
  - `50%`: 볼륨을 0%로 설정
  - `51% ~ 100%`: 볼륨을 50%로 설정
