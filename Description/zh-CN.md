该脚本为 YouTube Music 添加了音量调整快捷键。

## 使用方法

- `Alt + ↑`: 增加音量 1%
- `Alt + ↓`: 减少音量 1%
- `Ctrl + Alt + ↑`: 增加音量 5%
- `Ctrl + Alt + ↓`: 减少音量 5%
- `Ctrl + Shift + Alt + ↑`:
  - `0% ~ 49%`: 设置音量为 50%
  - `50%`: 设置音量为 100%
  - `51% ~ 100%`: 设置音量为 100%
- `Ctrl + Shift + Alt + ↓`:
  - `0% ~ 49%`: 设置音量为 0%
  - `50%`: 设置音量为 0%
  - `51% ~ 100%`: 设置音量为 50%
